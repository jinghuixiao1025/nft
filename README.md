## 介绍

该项目基于 Java 开发。

- 后端采用 Spring Boot、Spring Data Jpa、Sa Token、 Redis、Mysql。
- 前端采用 uniapp 跨端开发，一套代码，可发布到 iOS、Android、Web（响应式）等多个平台。
- 本项目主要供交流学习，请不要用于商业项目。
